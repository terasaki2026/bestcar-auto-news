
# BestCar Auto News (自動車ニュースまとめ)

自動車メーカー各社の最新ニュースを自動収集し、カード形式で一覧表示するダッシュボードアプリです。
RSSフィードや公式サイトのデータを取得し、Streamlitで美しく表示します。

## 📁 1. 準備：フォルダを開く

まず、このファイルがあるフォルダをエクスプローラーで開いてください。
場所: `c:\Users\terasaki\OneDrive - 株式会社　講談社ビーシー\デスクトップ\bestcar_auto_news　2026.2.15`

## 💻 2. 黒い画面（PowerShell）を開く

非エンジニアの方でも一番簡単な方法です。

1. エクスプローラーの上部にある「アドレスバー（パスが表示されている場所）」をクリックします。
2. 現在のパスが青く選択された状態になります。
3. そのままキーボードで `powershell` と入力し、`Enter` キーを押します。
4. 青い背景の「Windows PowerShell」というウィンドウが立ち上がります。

## 🐍 3. Pythonの確認

立ち上がったPowerShellの画面で、以下のコマンドを入力して `Enter` を押してください。
（`$` マークは入力不要です）

```powershell
python --version
```

- `Python 3.x.x` のように数字が表示されればOKです。
- エラーが出る場合は、Pythonがインストールされていません。Microsoft StoreなどでPythonをインストールしてください。

## 📦 4. 仮想環境の作成と有効化

システム全体に影響を与えないよう、このアプリ専用の「仮想環境」を作ります。

1. **仮想環境を作る**
   以下のコマンドを入力して `Enter` を押します（少し時間がかかります）。
   ```powershell
   python -m venv venv
   ```

2. **仮想環境に入る（有効化）**
   以下のコマンドを入力して `Enter` を押します。
   ```powershell
   .\venv\Scripts\Activate.ps1
   ```

   **⚠️ 重要：もし赤文字のエラーが出た場合**
   「このシステムではスクリプトの実行が無効になっているため…」というエラーが出たら、以下のコマンドを入力して `y` を押し、許可を与えてください。
   ```powershell
   Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```
   その後、もう一度 `.\venv\Scripts\Activate.ps1` を実行してください。
   成功すると、行の先頭に緑色の文字で `(venv)` と表示されます。

## 📥 5. ライブラリのインストール

必要な機能（Streamlitなど）を一括でインストールします。
以下のコマンドを入力して `Enter` を押します。

```powershell
pip install -r requirements.txt
```

白い文字や棒グラフがたくさん流れますが、最後に「Successfully installed...」と出れば完了です。

## 🚀 6. アプリの起動

準備完了です！以下のコマンドでアプリを起動します。

```powershell
streamlit run app.py
```

自動的にブラウザが立ち上がり、ニュースサイトが表示されます。
（初回はメールアドレスの入力を求められる場合がありますが、何も入力せずに `Enter` を押せばスキップできます）

---

- **終了したいとき**: PowerShellの画面で `Ctrlキー` を押しながら `C` を押すと停止します。
- **次回起動するとき**: 手順2（PowerShellを開く）→ 手順4-2（有効化）→ 手順6（起動） だけでOKです。
